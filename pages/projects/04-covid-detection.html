OP-004 // DEEP LEARNING

Predicting COVID with Convolutional Neural Networks

LIVE 2024.02
Python TensorFlow Deep Learning Computer Vision
Overview

COVID-19 (SARS-CoV-2) is a highly infectious disease that led to a global shutdown between 2020 and 2023. While rapid antigen testing is effective, it is costly, time-consuming, and requires specimen collection. This project proposes an image classification model using chest X-rays to distinguish patients with COVID-19, viral pneumonia, and normal lung presentation — with the potential to reduce testing costs, improve patient experience, and minimize HCP exposure to biohazards.

/// Github Repository — COVID Detection
Methods

Approach

Using chest X-ray data from Kaggle, provided by researchers at the University of Montreal, images and labels were extracted to train a convolutional neural network using TensorFlow. The model was trained on 217,859 parameters derived from 251 images, classifying X-rays into three categories: Normal, COVID-19, and Viral Pneumonia.

The training process used image augmentation techniques to improve the model's ability to generalize from visual features. The architecture comprised four convolutional layers with filter sizes of 256 and 64, kernel sizes of 7 and 3, and max pooling with 5×5 and 3×3 grids to reduce feature dimensions.

Despite strong classification accuracy in the first model, subsequent attempts to implement VGG16 and VGG19 architectures faced challenges in achieving stable learning.

Outcomes

Results

98%
COVID-19 Classification Accuracy
90%
Viral Pneumonia Accuracy
87%
Normal Case Accuracy

The CNN demonstrates a strong ability to classify chest X-rays across all three categories. The 98% accuracy for COVID-19 cases indicates high efficacy in identifying features associated with infection. The slightly lower pneumonia accuracy (90%) may reflect early or late-stage disease presentations that are harder to distinguish. Normal case accuracy (87%) may be affected by subtle variations in lung appearance from smoking, pre-existing conditions, or prior medical history.

Crucially, the model did not misclassify viral pneumonia as COVID-19, demonstrating meaningful feature separation between the two. Additional research with a larger, more diverse dataset would be needed before considering broader clinical applications.
